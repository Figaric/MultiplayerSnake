# Hub: /account

## RequestCreateUser

### Request

```json
{
    username: "",
    color: от 1 до 14
}
```

### Response

На клиенте вызывается метод ResponseCreateUser

```json
{
    ok: true or false,
    errors: [
        error: {
		field: "например: username",
    		message: "например: username is required."
        }
    ]
}
```

P.S. естественно если ошибок нет, то ``` errors: null```
